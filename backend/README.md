# Plan3 Backend API

这是 Plan3 在线销售微信小程序的后端 API 服务。

## 开发计划

- [ ] 设置 Node.js/Express 或 NestJS 框架
- [ ] 实现用户认证和授权
- [ ] 实现产品管理 API
- [ ] 实现订单管理 API
- [ ] 实现购物车 API
- [ ] 集成微信小程序 API
- [ ] 数据库设计和实现
- [ ] API 文档生成

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 生产环境启动
pnpm start
```

## API 文档

API 规范定义在根目录的 `api/openapi.yaml` 文件中。
